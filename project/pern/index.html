Research on Path-enhanced Explainable Recommendation with Knowledge Graphs

Sep 2, 2020

Recommender system helps us to obtain information/products that interest us more. Knowledge Graph (KG), composed of countless triples, is a wellstructured data form and has already used in recommendersystem, question answering, etc.

However, existing knowledge graph-based recommendation methods have several limitations. User-item path sets show poor explainablity and accuracy expecially in cold-start issuses. Therefore, we propose a novel model named Path-enhanced Recurrent Network (PeRN). Specifically, PeRN integrates a recurrent neural network encoder with a meta-path-based entropy encoder to further increase explainability, accuracy and reduce cold-start costs.

Recurrent network encoder could represent sequential path semantics in knowledge graph.

Entropy encoder is an efficient approach, which gather meta path information to differentiate paths in one user-item interaction. We also propose a path extraction algorithm with a bidirectional scheme,which is proposed to make PeRN more feasible.
